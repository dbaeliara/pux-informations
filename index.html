Activation et Désactivation de la Page Produit V8

Pour passer facilement de la page produit v8 (nouvelle version) à la page produit v7 (ancienne version), vous pouvez glisser-déposer ces 2 liens dans votre barre de bookmarks. Ce lien n'est utilisable que sur le site aramisauto, lors de la consultation d'une page produit.


Lien d'activation V7 : Activer V7
Lien d'activation V7 : Activer V8

Une fois les 2 bookmarks créés, vous pouvez aller sur une page produit et cliquer sur l'un des 2 bookmark. Le résultat est le passage en mode v8 ou v7 de la page, ce passage est persistant (pour toutes les autres pages produit).


Pour accéder à une page produit : Voir les modèles Peugeot sur le site.

Sur mobile : Créer un raccourci avec ce contenu:

javascript:location = location.href.replace(/&v8=./, '') + '&v8=1'
