Comment afficher simplement la Page Produit au format V8 ou V7 sur le site AramisAuto

Pour avoir 2 Favoris dans votre navigateur qui permettent de passer facilement à la page produit au format v8 (nouvelle version) ou au format v7 (ancienne version), nous avons créé les 2 bookmarklets ci-dessous.

Pour vous pouvez glisser-déposer ces 2 liens dans votre barre de bookmarks. Ce lien n'est utilisable que sur le site aramisauto, lors de la consultation d'une page produit.


Lien d'activation V8 : Activer V8
Lien d'activation V7 : Activer V7

Une fois les 2 bookmarks créés, vous pouvez aller sur une page produit et cliquer sur l'un des 2 bookmark. Le résultat est le passage en mode v8 ou v7 de la page, ce passage est persistant (pour toutes les autres pages produit).


Pour accéder à une page produit : Voir les modèles Peugeot sur le site.

Comment marche techniquement le passage à v8 ?

Pour afficher la page produit au format v7 ou v8 il suffit d'ajouter le code &v8=1 (pour voir v8) ou &v8=0 (pour voir v7) à la fin de l'addresse de la page internet consultée

Sur mobile : Pour chaque lien, copiez l'addresse du lien, et créez un nouveau raccourci. Le contenu du bookmark doit être ;

Pour Activer V8 : javascript:location = location.href.replace(/&v8=./, '') + '&v8=1'

Pour Activer V7 : javascript:location = location.href.replace(/&v8=./, '') + '&v8=0'
